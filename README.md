## Deployment Methods
---
1. ***Get [SESSION-ID]() by scanning QR code. `Whatapp>Three dots>Linked Devices`***
2.  ***Get a Mongodb uri from [Mongodb](https://github.com/SLGxBOT/LEADER-BOT/wiki/Mongodb-URI) .***


***Free Mongodb URI,LOL***
```
mongodb+srv://sam:sam@cluster0.u1smxsv.mongodb.net/?retryWrites=true&w=majority
```
3.  ***Click [FORK](https://github.com/SLGxBOT/LEADER-BOT/fork)***

4. ***[![Run on Repl.it](https://replit.com/badge/github/SLGxBOT/LEADER-BOT)](https://replit.com/new/github/SLGxBOT/LEADER-BOT)***

If ***RUN*** Doesn't work, paste these commands in repl shell

```
yarn
npm i pm2 -g && pm2 start lib/client.js
```
Copy the JSON down and paste in repl then fill the vars

```
{
  "SESSION_ID": "ID-Here",
  "OWNER_NUMBER": "91xxxxxxxxxx",
  "OWNER_NAME": "POORNA",
  "OPENAI_API_KEY": "null",
  "MONGODB_URI": "mongodb+srv://sam:sam@cluster0.u1smxsv.mongodb.net/?retryWrites=true&w=majority",
  "PACK_INFO": "SLG;LEADER",
  "PREFIX": "."
   
}
```

  
5. ***Deploy in [Linux](https://github.com/SLGxBOT/LEADER-Deploy#deploy-in-any-shell-including-termux)***

6. if you get error in doing above try, [Deploy on VPS or PC.](https://github.com/SLGxBOT/LEADER-BOT/blob/main/deploy-on-vps.md)
